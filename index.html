O que é Java?

Java é uma linguagem de programação orientada a objetos, amplamente utilizada em desenvolvimento de software, aplicações web, Android, e muito mais.

Exemplo de Código Java


public class olaMundo {
    public static void main(String[] args) {
        System.out.println("Ola, Mundo!");
    }
}

Curiosidades sobre Java

  • Java foi criado por James Gosling na Sun Microsystems.
  • É uma das linguagens mais populares do mundo.
  • O lema de Java é "Write Once, Run Anywhere" (Escreva uma vez, execute em qualquer lugar).
